# タスク完了時のチェックリスト

## コード変更後に必ず実行

### 1. フォーマット
```sh
yarn format
```

### 2. リント
```sh
yarn lint
```

### 3. ビルド確認
```sh
yarn build
```

### 4. テスト実行（テストがある場合）
```sh
yarn test
```

## Git操作

### コミット前
- コーディング規約に従っているか確認
- 不要なコメントやデバッグコードを削除
- TypeScriptのコンパイルエラーがないか確認

### Pull Request作成時
- 変更内容の説明を明確に記載
- 関連するissueがあれば参照
